# 🤖 AI Quote Generator Pro

An AI-powered web application that generates inspirational quotes using Large Language Models (LLMs) running locally with Ollama.

Built using **Flask**, **HTML**, **CSS**, **JavaScript**, and **Ollama (Llama 3.2)**.

---

# 🚀 Features

- ✨ AI-generated quotes
- 📚 Multiple categories
- 🤖 AI model selection
- ❤️ Favorite quotes
- 📋 Copy to clipboard
- 🖼 Download as PNG
- 📄 Download as PDF
- 🌙 Dark / Light Mode
- 📱 Responsive Design
- ⚡ Response Time Display
- 🎨 Glassmorphism UI
- 🎇 Animated Background
- 📤 Share to LinkedIn
- 📤 Share to X
- 📤 Share to WhatsApp
- 💾 Local Favorites Storage

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/AI-Quote-Generator.git
```

---

## Move into Project

```bash
cd AI-Quote-Generator
```

---

## Create Virtual Environment

Windows

```bash
python -m venv venv
```

Activate

```bash
venv\Scripts\activate
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Install Ollama

Download from:

https://ollama.com

---

## Pull Model

```bash
ollama pull llama3.2:3b
```

---

## Start Ollama

```bash
ollama serve
```

---

## Run Flask

```bash
python app.py
```

Open

```
http://127.0.0.1:5000
```
